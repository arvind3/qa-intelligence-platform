Engineering Perspective

QualiGraph delivers browser-native QA intelligence by combining deterministic analytics and semantic clustering over a unified schema.

Architecture Overview

UI (React/Vite) → Unified Schema Builder → DuckDB-WASM + SQLite-Vec → Embeddings/Clustering → Copilot + Charts.

Generate/Upload → Normalize entities/relationships → KPI queries + vector index → map clusters + traceability views.

Tech Stack

React 18 + ViteTypeScriptDuckDB-WASMsqlite-vec-wasmEChartsPlaywright + Vitest

How It Works

  1. Generate 10k unified schema records or upload JSON/CSV test data.
  2. Build canonical entities: requirements, tests, plans, suites, runs, defects, and link tables.
  3. Compute deterministic KPIs and distribution charts via DuckDB-WASM.
  4. Build semantic vectors in-browser and produce cluster maps.
  5. Enable governance via traceability gauges and downloadable schema artifacts.

Design Decisions

  • Local-first: no backend dependency for analysis.
  • Hybrid trust model: deterministic + semantic side by side.
  • Schema-first strategy for multi-tool integration.
  • Evidence artifacts generated per quality run.
